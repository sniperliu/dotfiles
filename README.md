# Setup my MacOS anywhere

Setup the MacOS everywhere, auto install the development tools mainly fish shell/iTerm/Emacs.

## Prerequisite

1. Git
1. Ruby

```shell
git clone https://github.com/sniperliu/dotfiles
cd dotfiles
./bootstrap.sh
```
